Mājaslapu izstrādātājs

Par mani

Es esmu augsti kvalificēts mājaslapu izstrādātājs ar pieredzi gan front-end, gan back-end izstrādē. Man ir izcilas prasmes dažādās programmēšanas valodās, tostarp HTML, CSS, JavaScript un PHP.

Es spēju izstrādāt responsīvas un lietotājam draudzīgas vietnes, kas atbilst klientu vajadzībām. Pietam regulāri sekoju līdzi jaunākajām mājaslapu izstrādes tendencēm un tehnoloģijām, un vienmēr meklēju jaunus izaicinājumus un iespējas uzlabot savas prasmes.

Izglītība

Liepājas Universitāte: Maģistra grāds datorzinātnēs.

Vidējā atzīme: 8.31

Studiju kursi: Datorvadības pamati, Datorgrafikas un attēlu apstrādes pamati, Datu bāzu vadības sistēmas, Risinājumu algoritmizēšana un programmēšana.

Prasmes

Programmēšanas valodas: HTML, CSS, JavaScript, PHP un Python

Datubāzes: MySQL, Oracle, Cloudscape, MongoDB

Front-end/back-end izstrāde: React, Node.js, Angular, Firebase

Valodas: Angļu, Vācu

Projekti

Responsīva e-veikala mājaslapa

  • Es izveidoju dizainu un izstrādāju responsīvu e-komercijas mājaslapu vietējam apģērba zīmolam, izmantojot programmēšanas valodas, HTML, CSS, JavaScript un PHP. Mājaslapā integrēju drošu pirkumu apmaksas un inventāra pārvaldības sistēmu.

Sociālo tīklu web lietotne

  • Izveidoju sociālo tīklu web lietotni ar Node.js, MongoDB un React, kas ļauj lietotājiem savā starpā sazināties un dalīties ar dažāda veida saturu. Izstrādāju lietotāju autentifikāciju un datu pārvaldību, izmantojot Firebase.

  • Lietotne saņēma pozitīvas atsauksmes no lietotājiem.

Tīmeklī bāzēts projektu pārvaldības rīks

  • Izstrādāju tīmeklī bāzētu projektu pārvaldības rīku, izmantojot React, JavaScript un Firebase, lai ļautu komandām sadarboties un efektīvi pārvaldīt uzdevumus. Izveidoju dažādas funkcijas, piemēram, uzdevumu prioritātes un komandas komunikāciju.

  • Rīks tika lieliski integrēts uzņēmumā, kas palielināja komandas produktivitāti un projektu efektivitāti.
